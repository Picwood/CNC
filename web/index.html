4th-Axis Pocket Generator

Wrapped oblong pockets, now as a live web surface.

This demo turns the existing cylindrical oblong pocket logic into an embeddable browser component. It keeps the shaft/pocket parameterization from the Python tool and shows the result directly on a 3D cylindrical mesh so users can tune the model in real time.

Embed

Drop the module on any page and place the custom element where you want the configurator to appear. 

<script type="module" src="./cylindrical-oblong-pocket-viewer.js"></script>
<cylindrical-oblong-pocket-viewer></cylindrical-oblong-pocket-viewer>
